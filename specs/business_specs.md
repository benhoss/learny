
# Business Specifications — AI School Learning App

## 1. Vision
Build an AI-powered learning companion that helps children truly understand school subjects through active learning and short, engaging sessions.

The product aims to teach children **how to learn with AI**, not how to cheat.

---

## 2. Target Users

### Primary
- Children aged 10–14 (upper primary / lower secondary)

### Secondary
- Parents looking for efficient and trustworthy learning support

---

## 3. Problem Statement
- School learning is often passive and based on memorization
- Children lack effective study methods
- AI tools are misused for shortcuts instead of understanding

---

## 4. Value Proposition
- Learning from the child’s *real school material*
- Personalized, adaptive learning paths
- Short, efficient sessions (3–10 minutes)
- Ethical and pedagogical use of AI

---

## 5. User Journeys

### Parent Journey
1. Create account
2. Create child profile
3. Upload school material
4. Monitor progress and weaknesses

### Child Journey
1. Upload or select learning material
2. Play mini-games to learn concepts
3. Receive instant feedback
4. Use “Revision Express” before tests

### Onboarding Focus (New Priority)
- Child-first onboarding must be easy, fun, and engaging, with a first learning action in under 90 seconds.
- Parent onboarding must support creating and supervising one or more child accounts.
- Detailed onboarding recommendations: see `specs/onboarding_research_spec.md`.
- Implementation PRD for execution scope and rollout: see `specs/onboarding_implementation_prd.md`.
- Scan-first onboarding strategy (start with homework scan before account creation): see `specs/scan_first_onboarding_spec.md`.

---

## 6. Core Features (MVP)

- Parent & child accounts
- School material upload (photo / PDF)
- AI-generated learning packs
- Mini-games:
  - Flashcards
  - Quizzes
  - Matching
- Revision Express mode
- Smart notifications (push + email) with parent controls

---

## 7. Success Metrics (KPIs)

- Weekly active learners
- Session completion rate
- Retention (D7 / D30)
- Mastery progression per concept
